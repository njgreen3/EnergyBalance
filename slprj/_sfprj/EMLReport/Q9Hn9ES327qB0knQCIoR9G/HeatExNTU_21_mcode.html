1function [ P, Tho, Tco ] = HeatExNTU_2( Thi, m_dot_h, p_h, Tci, m_dot_c, p_c, fluid_h, ...
  2fluid_c, A, U, HeatExType)
  3%HeatExNTU Summary 
  4%   This function calculates flow and outlet temperatures of a heat exchanger
  5%   using the Number of Transfer Units (NTU) method as described in
  6%   Fundementals of Heat and Mass Transfer by F. P. Incropera, D. P.
  7%   DeWitt, T. L. Bergman & A. S. Lavine. The method can be used when only
  8%   the inlet tempertures are known as opposed to inlet and outlet
  9%   temperatures.
 10%
 11%   This is done by calculating effectiveness which is defined as the ratio
 12%   of actual heat transferred to the maximum possible heat transferred for 
 13%   an infinite length counter-flow heat exchanger with the given inlet
 14%   temperatures. The effectivenes is calculated differently depending on 
 15%   the geometry of exchanger but is always a function of the ratio of heat
 16%   capacity rates as well as the number of transfer units (NTU). The NTU
 17%   depends on the overall heat transfer coefficient, the heat transfer
 18%   area, and the smaller of the two heat capacity rates. 
 19%   
 20%   In the event of a phase change the rate of change of energy required to
 21%   get the fluid just beyond the point of vaporizaion or condensation is
 22%   calculated using the CoolProp function. Then the function is called
 23%   again at the new temperature and the final temperatures and heat rates
 24%   are calucluated using a reduced heat exhange area. I currently assume
 25%   the effected heat transfer area is a half of the original input. This 
 26%   function cannot accurately handle both fluids changing phase because it
 27%   does not check which fluid would change first.
 28% 
 29% Note: The effectiveness is separte from efficiency. The calculations
 30%   assumes an efficiency of 1 where all heat is transferred from the hot
 31%   fluid to the cool fluid or remains within the the fluid. This also
 32%   assumes no fluid is undergoing a change in state, which may not be
 33%   sufficient for my final model.
 34%
 35% This function makes use of the CoolProp wrapper for fluid properties. 
 36% The high level Matlab wrapper cannot handle vector inputs. There ways to
 37% make low level calls interfacing with the DLL so many values can be
 38% determined more quickly, but that is not implented here.
 39% More information about the Matlab wrapper can be found at
 40%   http://www.coolprop.org/coolprop/wrappers/MATLAB/index.html
 41% A list of parameters can be found at 
 42%   http://www.coolprop.org/coolprop/HighLevelAPI.html#parameter-table
 43% A list of fluids can be found at 
 44%   http://www.coolprop.org/fluid_properties/PurePseudoPure.html#list-of-fluids
 45%
 46% Inputs:
 47%   Thi         Hot fluid inlet temperature in Kelvin
 48%   m_dot_h     Hot fluid mass flow rate in kg/s
 49%   p_h         Hot fluid pressure in pa
 50%   fluid_h     Name of Hot fluid (string)
 51%   Tci         Cool fluid inlet temperature in Kelvin
 52%   m_dot_c     Cool fluid mass flow rate in kg/s
 53%   p_c         Cool fluid pressure in pa
 54%   fluid_c     Name of Cool fluid (string)
 55%   HeatExType  String describing the heat exchanger geometry. Currently
 56%               this variable only accepts 'counter' and 'parallel', but
 57%               can be expanded to other exchanger/flow types
 58%               'counter' for counter flow
 59%               'parallel' for parallel
 60%   U           Overall heat transfer coefficient in W/(K m2)
 61%   A           Heat transfer area in m2
 62%
 63% Output:
 64%   P           Rate of heat exchenged (power) in W
 65%   Tho         Hot fluid outlet temperature in input temp units (K)
 66%   Tco         Cool fluid outlet temperature in input tempt units (K)
 67
 68% initailize certain parameters used to call coolprop library
 69buffer_size = 1000;
 70ierr = 0;
 71b = (1:1:buffer_size);
 72herr= char(b);
 73backend = 'HEOS';
 74
 75% All the input signals should be the same size, so it shouldn't matter
 76% which one is used to obtain signal length
 77var_length = length(Thi);
 78
 79% initialize handles for hot and cool fluids
 80[handle_h, ~] = calllib('coolprop','AbstractState_factory',backend,fluid_h,ierr,herr,buffer_size);
 81[handle_c, ~] = calllib('coolprop','AbstractState_factory',backend,fluid_c,ierr,herr,buffer_size);
 82
 83% determine input parameter pair indecies for coolprop calls
 84[pq_ind,~] = calllib('coolprop','get_input_pair_index','PQ_INPUTS'); %pressure and Mass vapor quality
 85[pt_ind,~] = calllib('coolprop','get_input_pair_index','PT_INPUTS'); %pressure and temperature
 86[hp_ind,~] = calllib('coolprop','get_input_pair_index','HmassP_INPUTS'); %Mass specific enthalpy and pressure
 87
 88% determine output parameter indecies for coolprop calls
 89[t_ind, ~] = calllib('coolprop','get_param_index','T'); %temperature
 90[h_ind, ~] = calllib('coolprop','get_param_index','Hmass'); %Mass specific enthalpy
 91[c_ind, ~] = calllib('coolprop','get_param_index','Cpmass'); %Mass specific constant pressure specific heat
 92
 93% find pointer location of pressure and temperature input values
 94p_h_Ptr = libpointer('doublePtr',p_h);
 95p_c_Ptr = libpointer('doublePtr',p_c);
 96Thi_Ptr = libpointer('doublePtr',Thi);
 97Tci_Ptr = libpointer('doublePtr',Tci);
 98
 99% Determine vaporization temperature for fluids at respective pressures.
100% Initialize pointer locations for relevant variables
101Q_Ptr = libpointer('doublePtr', zeros(var_length)); % Q is specifically set to 0
102Tvh_Ptr = libpointer('doublePtr', zeros(var_length)); % initialized to 0 temporarily
103Tvc_Ptr = libpointer('doublePtr', zeros(var_length)); % initialized to 0 temporarily
104
105% Update values located at the Tvc & Tvh pointer locations for vaporization
106% conditions
107calllib('coolprop','AbstractState_update_and_1_out', handle_h, pq_ind, p_h_Ptr, Q_Ptr, var_length, t_ind, Tvh_Ptr, ierr,herr,buffer_size);
108calllib('coolprop','AbstractState_update_and_1_out', handle_c, pq_ind, p_c_Ptr, Q_Ptr, var_length, t_ind, Tvc_Ptr, ierr,herr,buffer_size);
109
110% Set Tvh & Tvc to values stored at pointer location
111Tvh = get(Tvh_Ptr,'Value');
112Tvc = get(Tvc_Ptr,'Value');
113
114% Determine the specific heats of the incoming fluids using coolprop
115% Initialize pointer locations for relevant variables
116Ch_Ptr = libpointer('doublePtr', zeros(var_length)); % initialized to 0 temporarily
117Cc_Ptr = libpointer('doublePtr', zeros(var_length)); % initialized to 0 temporarily
118
119% Update values located at the Tvc & Tvh pointer locations for vaporization
120% conditions
121calllib('coolprop','AbstractState_update_and_1_out', handle_h, pt_ind, p_h_Ptr, Th_Ptr, var_length, c_ind, Ch_Ptr, ierr,herr,buffer_size);
122calllib('coolprop','AbstractState_update_and_1_out', handle_c, pt_ind, p_c_Ptr, Tc_Ptr, var_length, c_ind, Cc_Ptr, ierr,herr,buffer_size);
123
124% Set Tvh & Tvc to values stored at pointer location
125Ch = get(Ch_Ptr,'Value');   % J/(kg K)
126Cc = get(Cc_Ptr,'Value');   % J/(kg K)
127
128% If the fluid is withing 1e-4 units of the vaporization point then the 
129% will return an inaccurate values. If that happens then the specific heat 
130% will be set to inf so its temperature will not change even though heat is
131% exchanged.
132Ch(abs(Thi - Tvh) < 1e-4) = inf;    % J/(kg K)
133Cc(abs(Tci - Tvc) < 1e-4) = inf;    % J/(kg K)
134
135% Calculate the heat capacity rates of the hot and cool fluids
136Ch_rate = Ch .* m_dot_h;     % W/K
137Cc_rate = Cc .* m_dot_c;     % W/K
138
139% Determine the maximum and minimum of the two heat capacity rates
140Cmax = Ch_rate; % W/K
141Cmax(Ch_rate < Cc_rate) = Cc_rate(Ch_rate < Cc_rate);   % W/K
142Cmin = Ch_rate; % W/K
143Cmin(Ch_rate > Cc_rate) = Cc_rate(Ch_rate > Cc_rate);   % W/K
144
145
146% Calcluate the heat capacity rate ratio 
147Cr = Cmin./Cmax;
148
149% Calculate the number of transfer units of the heat exchanger
150NTU = U * A ./ Cmin;
151
152% Calculate effectiveness based on type of heat exchanger
153switch HeatExType
154    case 'counter'  % If the heat exchanger is counter-flow
155        effectiveness = (1 - exp(-NTU.*(1 - Cr))) ./ ...
156                        (1 - Cr.*exp(-NTU.*(1 - Cr)));
157    case 'parallel' % If the heat exchanger is parallel-flow
158        effectiveness = (1 - exp(-NTU.*(1 + Cr))) ./ ...
159                        (1 + Cr);
160    otherwise       % If not exchanger specified
161        warning('No heat exchanger type specified, therefore no heat is exchanged.')
162        effectiveness = 0;
163end        
164
165% % However if the Cr is 0 (Cmax is inf) then effectiveness is calcluated
166% % regardless of the heat exchanger flow.
167% effectiveness(Cr == 0) =  1 - exp(-NTU(Cr == 0));
168
169% Maximum energy transfer rate is equal for an infinite length heat
170% exchanger is equal to Cmin multiplied with the difference in inlet
171% temperatures. The actual energy transfer rate is the maximum rate
172% multiplied by the effectiveness (Assuming no heat loss).
173P = effectiveness.*Cmin.*(Thi - Tci);
174
175% The change in temperature is calculated for both sides by dividing the
176% energy tranfer rate by the heat capacity rates for both sides. This
177% assumes the energy transfer rate is the same for both sides.
178deltaTh = P./Ch_rate;
179deltaTc = P./Cc_rate;
180
181% The outlet temperatures are caluctaed by subtracting the change in
182% temperature from the hot fluid inlet temperature and adding the change in
183% temperature from the cool fluid inlet temperature.
184Tho = Thi - deltaTh;
185Tco = Tci + deltaTc;
186    
187% check if hot fluid would condense 
188if Tho < Tvh && Thi > Tvh
189%    disp('Condensation')
190%  Determine the enthalpy extracted from the hot fluid to condense it from 
191%  starting temeprature. A slight 1e-4 K temperature change beyond
192%  condensation is subtracted because coolprop cannot handle fluids within 
193%  1e-4 K of the vaporization/condensation curve.
194   Hh_init = CoolProp.PropsSI('H', 'T', Thi, 'P', p_h, fluid_h);
195   Hh_cond = CoolProp.PropsSI('H', 'T', Tvh - 1e-4, 'P', p_h, fluid_h);
196      
197%  Calculate rate of heat transfer due to change in temp and condensation
198   P_cond = (Hh_init - Hh_cond).*m_dot_h;
199   
200%  recalucluate the change in temp for the cool fluid due to condensation
201%  of the hot fluid
202   deltaTc = P_cond./Cc_rate;
203%  recalculate the temperature of the cool fluid 
204   Tc_cond = Tci + deltaTc;
205   
206%  Run this function again but with the input temps set to temps after
207%  condensation. Because the caluclation for the heat exchanger is broken
208%  up to multiple parts, the effective area should be reduced. I will
209%  temporarily assume it is reduced by a factor of two but should return to
210%  this later.
211   [ P_deltaT, Tho, Tco ] = HeatExNTU_2( Tvh - 1e-4, m_dot_h, p_h, fluid_h, ...
212       Tc_cond, m_dot_c, p_c, fluid_c, HeatExType, U, A/2 );
213   P = P_cond + P_deltaT;
214
215% else check if cool fluid would vaporize
216elseif Tco > Tvc && Tci < Tvc
217%    disp('Vaporization')
218%  Determine the enthalpy needed to vaporize the cool fluid from 
219%  starting temeprature. A slight 1e-4 K temperature change beyond
220%  vaporization is added because coolprop cannot handle fluids within 
221%  1e-4 K of the vaporization/condensation curve.
222   Hc_init = CoolProp.PropsSI('H', 'T', Tci, 'P', p_c, fluid_c);
223   Hc_vap = CoolProp.PropsSI('H', 'T', Tvc + 1e-4, 'P', p_c, fluid_c);
224      
225%  Calculate rate of heat transfer due to change in temp and vaporization
226   P_vap = (Hc_vap - Hc_init).*m_dot_c;
227   
228%  recalucluate the change in temp for the hot fluid due to vaporzation of
229%  the cool fluid
230   deltaTh = P_vap./Ch_rate;
231%  recalculate the temperature of the hot fluid 
232   Th_vap = Thi - deltaTh;
233   
234%  Run this function again but with the input temps set to temps after
235%  vaporization. Because the caluclation for the heat exchanger is broken
236%  up to multiple parts, the effective area should be reduced. I will
237%  temporarily assume it is reduced by a factor of two but should return to
238%  this later.
239   [ P_deltaT, Tho, Tco ] = HeatExNTU_2( Th_vap, m_dot_h, p_h, fluid_h, ...
240       Tvc + 1e-4, m_dot_c, p_c, fluid_c, HeatExType, U, A/2 );
241   P = P_vap + P_deltaT;
242end
243
244% check if hot fluid would condense and cool fluid would vaporize
245if (Tho < Tvh && Thi > Tvh) && (Tco > Tvc && Tci < Tvc)
246    error('This function cannot accurately handle both fluids changing phase.')
247end
248
249end
